dkiscm.excelie Installation
---------------------------

To install dkiscm.excelie using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``dkiscm.excelie`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        dkiscm.excelie

* Re-run buildout, e.g. with:

    $ ./bin/buildout
